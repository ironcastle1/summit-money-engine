# 30k-line build plan

This is being built as a real system, not as repeated single-file mockups.

## Part 1 — Map-first signal terminal
Target: 4k-7k lines.
- Backend API
- Map-first frontend
- Hidden side panels
- Live market data
- Polymarket feed
- Signal scoring rules
- Map nodes and trade routes
- Verification gates

## Part 2 — Data engine expansion
Target: +6k-8k lines.
- Persistent database
- Price snapshots
- Signal history
- Event-market history
- More market APIs
- Sector mappers
- News deduplication
- Source credibility scoring

## Part 3 — Backtesting and proof engine
Target: +6k-8k lines.
- Signal backtester
- Forward performance tracker
- Watchlist simulator
- False-positive database
- Do-nothing filter
- P/L model by asset route

## Part 4 — User product layer
Target: +6k-8k lines.
- Accounts
- Billing gate
- Saved signals
- Alerts
- Morning money briefing
- Admin panel
- Subscription management

## Part 5 — Pro terminal layer
Target: +6k-10k lines.
- WebSocket updates
- Advanced charts
- Market heatmaps
- Country and route overlays
- Custom watchlists
- API keys
- Export/reporting

Minimum combined target: 30k+ lines.
